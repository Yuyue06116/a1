The 'brain lesions' file documents somatic symptom-related brain lesions extracted from published studies, traced in 3D Slicer and registered to the MNI152 T1 2 mm template.
The 'brain stimulation coordinates' file documents effective brain stimulation coordinates for somatic symptoms extracted from published research.
The 'interoceptive coordinates' file documents interoceptive processing coordinates extracted from published research.
The 'normative modeling correlation analysis' file documents the correlation analysis between personalized total atrophy voxel number and PHQ-15 scores.
The 'somatic symptom brain imaging coordinates' file documents multimodal neuroimaging abnormal coordinates (ALFF, ReHo, GMV, task fMRI) of somatic symptoms from published research
The 'somatic_sens80_spec_zFCmap' file is the somatic network derived from lesion network mapping.
